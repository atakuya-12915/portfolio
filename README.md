# NAGOYAMESHI

## 📝 概要
食べログ風のレストラン検索・レビュー投稿アプリです。

## 🎯 制作背景
「ユーザーが飲食店を探しやすく、信頼できるレビューを投稿できる仕組み」を目指して開発しました。

## 🛠️ 使用技術
- Java 17
- Spring Boot 3.2.0
- MySQL 8.0
- Thymeleaf
- Spring Security

## ✨ 実装機能
- 会員登録・ログイン認証
- レストラン検索（エリア・価格帯での絞り込み）
- レビュー投稿・編集・削除
- お気に入り登録

## 🔧 工夫した点
- MyBatisでの複雑なJOIN処理
- Spring Securityでのロール別アクセス制御
- Thymeleafでの動的UI生成
